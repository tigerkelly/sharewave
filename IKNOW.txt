I normally setup up the drive first, if using a new SSD or SAS drive.

I normally execute the ./build.sh and then the sudo ./install.sh

The gui requires ask for a new password to manage the gui the very firat time you execute it.

If this is your first time running the sharewave.server then you need to execute it by hand so that you can
supply the password when asked.

The sharewave-gui program will ask you to create a password for the sharewave-gui program.

When you click on to the connect button in the gui it will ask you for the password you gave to the server.

The mount point will need is permissions set, do this before you execute the server for the first time.
    shdo chmod 0777 /your_mount_point

The server will create the additional dirtectories that are needed.
